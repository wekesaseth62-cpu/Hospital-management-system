# Hospital Management System for Rural Clinics

A Python and SQLite based hospital management system designed to improve patient record management and appointment scheduling in small healthcare centers.

## Features

• Patient registration and record management  
• Staff management  
• Appointment scheduling  
• Search and update functions  
• Commit history tracking  
• Input validation  

## Technologies Used

• Python  
• SQLite3  
• Datetime module  
• Pydroid 3  

## Inspiration

This project was inspired by healthcare challenges I observed while helping manage responsibilities at home and within my community. I wanted to reduce delays caused by manual record keeping and improve efficiency in clinics with limited resources.

I developed much of this project using a Redmi A5 smartphone and Pydroid 3, often planning program logic by hand before testing the code digitally.

## Challenges Faced

One major challenge involved debugging repeated database entries caused by recursive loop errors. I used documentation, AI assisted debugging tools, and manual testing to trace and rebuild the system logic.
